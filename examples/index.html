Quick Start

Get up and running in seconds

index.html 
<!-- 1. Include CSS -->
<link rel="stylesheet" href="nepali-datepicker.css">

<!-- 2. Add container -->
<div id="myPicker"></div>

<!-- 3. Include JS -->
<script src="nepali-datepicker.js"></script>

<!-- 4. Initialize -->
<script>
  NepaliDatePicker.init('#myPicker', {
    mode: 'bilingual',
    theme: 'default',
    onChange: function(date) {
      console.log(date);
    }
  });
</script>

Configuration

All available options

Option Type Default Description
mode String 'bilingual' 'bilingual', 'nepali', or 'english'
theme String 'default' 'default', 'ocean', 'forest', 'sunset', 'rose'
dark Boolean false Enable dark mode
minYear Number 2070 Minimum BS year
maxYear Number 2099 Maximum BS year
closeOnSelect Boolean true Close after selection
showToday Boolean true Show Today button
showClear Boolean true Show Clear button
onChange Function null Callback on date select

API Methods

Programmatic control

api.js 
// Initialize
const picker = NepaliDatePicker.init('#picker', opts);

// Get date
picker.getDate();  // { bs: {...}, ad: {...} }

// Set date
picker.setDate(2081, 5, 15);

// Open / Close / Toggle
picker.open();
picker.close();
picker.toggle();

// Clear selection
picker.clear();

// Change theme
picker.setTheme('ocean');
picker.setDark(true);

// Utility functions
NepaliDatePicker.utils.bsToAd(2081, 5, 15);
NepaliDatePicker.utils.adToBs(2024, 8, 30);
NepaliDatePicker.utils.toNepali(2081);  // '२०८१'
NepaliDatePicker.utils.getToday();
Calendar Data Range
Accurate BS data included from 2070 to 2099. Dates outside this range may not convert correctly.
